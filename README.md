# ESP-Mindiff-App

Une application Flutter pour ESP-Mindiff.

## 🚀 Démarrage

### Prérequis
- Flutter SDK installé (version 3.24.5 ou supérieure)
- Dart SDK 3.5.4 ou supérieure

### Installation

1. Récupérer les dépendances :
```bash
flutter pub get
```

2. Lancer l'application :
```bash
flutter run
```

### Plateformes supportées
- 📱 iOS
- 🤖 Android
- 🖥️ macOS
- 🌐 Web
- 🪟 Windows
- 🐧 Linux

## 📁 Structure du projet

```
lib/
  └── main.dart          # Point d'entrée principal de l'application
pubspec.yaml             # Configuration des dépendances et métadonnées
test/                    # Tests unitaires et de widgets
```
